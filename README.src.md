# prototag
**Go style tag parser for protocol buffers**

## Install

for cli  
```go install github.com/wirekang/prototag/cmd/prototag@latest```

for go modules   
```go get github.com/wirekang/prototag```

## Example

```protobuf
##PROTO##
```
